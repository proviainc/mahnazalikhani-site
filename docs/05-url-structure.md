# Stage 05 - URL Structure

## URL Principles

- Keep URLs short, readable, and durable.
- Use lowercase kebab-case.
- Use trailing slashes only if the chosen framework/site convention uses them consistently.
- Do not create search pages without enough unique content.
- Avoid duplicating ProVia Events service URLs on Mahnaz's personal site.
