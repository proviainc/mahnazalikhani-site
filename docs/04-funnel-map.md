# Stage 04 - Funnel Map

## Funnel Principle

The site should guide visitors from trust validation to one clear action: booking an Event Strategy Consultation.

## Awareness Stages

| Awareness Stage | Visitor Mindset | Content Needed | CTA Intensity |
|---|---|---|---|
| Unaware / low awareness | "Who is Mahnaz?" | Hero, bio, proof strip, LinkedIn | Low to medium |
| Problem-aware | "My event has a lot of moving parts." | Problem framing, event risks, where she helps | Medium |
| Solution-aware | "I need senior event strategy or execution support." | Offer framing, selected experience, consultation coverage | Medium to high |
| Decision-ready | "Can I trust her and book?" | Proof, fit, direct CTA, Calendly | High |

## Primary Journey: Consultation Booking

1. Visitor lands on homepage.
2. Hero clarifies Mahnaz's role and value.
3. Proof strip shows verified experience.
4. Audience-fit section confirms relevance.
5. Operating philosophy explains how she thinks.
6. Selected experience validates scale.
7. Consultation section explains what the call covers.
8. Visitor clicks Calendly.

## Corporate / Business Event Journey

Need:

- Senior judgment, stakeholder confidence, room flow, brand alignment, and execution reliability.

Recommended page path:

- Homepage -> Event Strategy section/page -> Selected Experience -> Calendly.

Trust builders:

- Corporate/event examples.
- Guest flow and vendor coordination language.
- ProVia Events context.
- EventAgent systems perspective.

## Luxury Private Event Journey

Need:

- Discretion, warmth, calm coordination, and polished host/guest experience.

Recommended page path:

- Homepage -> About/Experience -> Event Strategy -> Calendly.

Trust builders:

- Hospitality judgment.
- Privacy-aware copy.
- Avoid overexposure of private events.
- One strong portrait and restrained proof.

## Hospitality Consulting Journey

Need:

- Practical operational improvement, service flow, staff readiness, and guest experience.

Recommended page path:

- Homepage -> About -> Experience -> Calendly.

Trust builders:

- Hospitality Management education.
- Restaurant/banquet hall consulting.
- Banic Software hospitality-sector background.

## ProVia Events Validation Journey

Need:

- Understand Mahnaz's personal credibility behind ProVia Events.

Recommended page path:

- Homepage -> About -> ProVia Events context -> Calendly or ProVia Events outbound.

Trust builders:

- Clear statement that ProVia Events has its own company website.
- Mahnaz's leadership and selected event experience.

## EventAgent Validation Journey

Need:

- Understand Mahnaz's event operations and strategy credibility.

Recommended page path:

- Homepage -> EventAgent context -> LinkedIn or EventAgent outbound.

Trust builders:

- Systems and technology perspective.
- Vendor/timeline/approval language.
- Exact EventAgent title once approved.

## Speaking / Emcee Journey

Need:

- Verify public presence, stage comfort, and event leadership.

Recommended page path:

- Homepage -> Experience -> About -> Calendly or LinkedIn.

Launch recommendation:

- Keep this as a secondary proof path unless the user later decides speaking/emcee inquiries are a primary business goal.
