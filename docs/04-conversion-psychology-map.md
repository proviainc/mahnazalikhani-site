# Stage 04 - Conversion Psychology Map

## Conversion Principle

The site should persuade through clarity, proof, and guided next steps. It should not rely on pressure, generic urgency, or inflated event-planning promises.

## Core Levers

| Lever | How It Applies | Website Expression |
|---|---|---|
| Certainty | Buyers need confidence that Mahnaz can handle complex event realities. | Clear positioning, proof metrics, selected experience, consultation explanation. |
| Authority | Mahnaz needs senior credibility without sounding arrogant. | 10+ years, 18 years total experience, 100+ events, large event counts, Banic Software, ProVia Events, EventAgent context. |
| Risk reduction | High-stakes event buyers fear embarrassment, guest confusion, vendor issues, and timeline drift. | Problem framing, operating philosophy, proof, what the consultation covers. |
| Specificity | Generic event promises are overused. | Name guest flow, vendor coordination, sponsor alignment, timeline control, stage flow, hospitality operations. |
| Simplicity | Visitors should not have to choose among too many paths. | One primary CTA, one secondary LinkedIn path, limited contextual outbound links. |
| Guided next step | Visitors should know exactly what to do after trust is established. | Repeated Calendly CTA with clear labels and supporting explanation. |
| Objection neutralization | Visitors may wonder whether to contact Mahnaz, ProVia Events, or EventAgent. | Clarify the role of each brand in separate contained sections. |
| Loss aversion | Delaying strategy can create event-day confusion and avoidable risk. | Frame the cost of unclear guest flow, vendor roles, timelines, and program handoffs without fearmongering. |
| Social proof | Current proof is mostly experience-based, not testimonials. | Use verified metrics and LinkedIn until testimonials/assets are approved. |
| Speed | High-intent visitors need a fast route to action. | Header CTA and Calendly link visible early. |

## Psychological Flow For Homepage

1. Clarity: "This is Mahnaz, and this is her role."
2. Relevance: "She helps people like me."
3. Certainty: "She has handled meaningful event complexity before."
4. Specificity: "She understands the real moving parts I worry about."
5. Trust: "Her background spans hospitality, execution, business, and systems."
6. Guided action: "Book an Event Strategy Consultation."

## What To Avoid

- Fake urgency.
- Countdown timers.
- Overpromising outcomes.
- Generic "stress-free" language.
- Too many CTA choices.
- Unverified testimonials or logos.
- Making EventAgent feel like the main conversion path.

## Copy Implications For Stage 06

Stage 06 should write copy that:

- Names event risks concretely.
- Uses proof metrics early.
- Explains what the consultation covers.
- Reassures visitors that ProVia Events and EventAgent are related but not the main site identity.
- Presents booking as a practical next step, not a hard sell.
