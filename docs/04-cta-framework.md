# Stage 04 - CTA Framework

## CTA Hierarchy

### Primary CTA

Label:

- Book an Event Strategy Consultation

Destination:

- `https://calendly.com/mahnazalikhani/event-strategy`

Use:

- Header
- Hero
- Event Strategy section
- Selected Experience section
- Final CTA band
- Footer

### Secondary CTA

Label:

- Connect on LinkedIn

Destination:

- `https://www.linkedin.com/in/mahnazalikhani/`

Use:

- Hero as secondary link
- About section
- Footer
- Mobile menu

### Contextual CTA: ProVia Events

Label:

- Visit ProVia Events

Destination:

- `https://proviaevents.com/`

Use:

- Only in a ProVia Events context section.

Purpose:

- Route visitors who need the company-service website.

### Contextual CTA: EventAgent

Label:

- Learn About EventAgent

Destination:

- `https://eventagent.io/`

Use:

- Only in a contained EventAgent context section.

Purpose:

- Support Mahnaz's event-operations and strategy credibility without turning the page into a product pitch.

## CTA Placement Rules

- The first visible CTA should be Calendly.
- Never place ProVia Events or EventAgent CTAs above the primary Calendly CTA.
- Avoid using "Contact" as the main button unless it routes to the consultation.
- Use LinkedIn as a trust path, not as a competing primary action.
- Repeat the Calendly CTA after proof and near the page end.

## CTA Copy Variants

Use:

- Book an Event Strategy Consultation
- Discuss Your Event With Mahnaz
- Book a Strategy Call

Use carefully:

- Plan Your Event
- Start Your Event Strategy
- Work With Mahnaz

Avoid:

- Get Started
- Submit
- Contact Us
- Free Consultation, unless the call is explicitly free
- Book Now, if it feels too transactional
