# Stage 05 - Trust-Link Placement

## Trust-Link Strategy

External links should support trust without pulling visitors away before the site has made the case for booking a consultation.

## LinkedIn

Role:

- Primary social verification link.

Placement:

- Header or mobile menu as secondary link/icon.
- Hero secondary CTA.
- About page near bio.
- Footer.

Label options:

- Connect on LinkedIn
- View LinkedIn
- LinkedIn

## ProVia Events

Role:

- Company-service context and credibility anchor.

Placement:

- Homepage section explaining Mahnaz's ProVia Events context.
- About page career/affiliation section.
- Footer trust/context links.

Label options:

- Visit ProVia Events
- Learn About ProVia Events

Rules:

- Do not place in primary header navigation.
- Do not make it visually equal to the Calendly CTA.
- Do not duplicate ProVia Events service pages on Mahnaz's site.

## EventAgent

Role:

- Event-operations and strategy credibility anchor.

Placement:

- Homepage contextual section.
- About page career/strategy section.
- Footer trust/context links.

Label options:

- Learn About EventAgent
- Visit EventAgent

Rules:

- Do not lead with EventAgent unless the visitor is in the EventAgent context section.
- Do not overstate Mahnaz's title until approved.
- Keep copy focused on event operations, systems, vendors, timelines, and coordination.

## Calendly

Role:

- Primary conversion destination.

Placement:

- Header CTA.
- Hero CTA.
- Event Strategy page.
- End of major pages.
- Footer.

Label:

- Book an Event Strategy Consultation

## Placement Priority

1. Calendly: visible early and repeated.
2. LinkedIn: visible but secondary.
3. ProVia Events: contextual.
4. EventAgent: contextual.

## Anti-Patterns

Avoid:

- A header full of external links.
- ProVia Events and EventAgent CTAs competing with booking.
- LinkedIn as the only meaningful action.
- External links before Mahnaz's value is clear.
