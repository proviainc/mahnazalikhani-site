## Component Rules

### Header

- Primary CTA visible.
- No ProVia Events/EventAgent primary nav items.
- Mobile menu includes Book Consultation and LinkedIn.

### Button

- Primary: Calendly actions only.
- Secondary: LinkedIn or contextual links.
- No generic "Get Started."

### Proof Strip

- No animated counters.
- Use verified metrics only.
- Must stack cleanly on mobile.

### Experience Cards

- Must support text-only proof.
- No unapproved images.
- Include event type, scale, and responsibility.

### Brand Context Block

- ProVia Events and EventAgent links must be visually subordinate to Calendly.
