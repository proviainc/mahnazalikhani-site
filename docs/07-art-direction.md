# Stage 07 - Art Direction

## Visual Territory

Territory:

- Editorial personal brand with event-operations discipline.

The site should feel like a composed profile of a senior strategist, not a gallery-driven event planner site.

## Photography Direction

### Primary Portrait

Use a portrait that feels:

- Warm.
- Confident.
- Professional.
- Calm.
- Direct.

Preferred settings:

- Neutral indoor environment.
- Hospitality, venue, office, or event-adjacent setting.
- Natural or soft studio lighting.

Avoid:

- Casual selfies.
- Party/event chaos in the background.
- Heavy filters.
- Overly staged luxury poses.
- Crops that make Mahnaz feel distant or inaccessible.

## Supporting Imagery

Use supporting event imagery only if approved.

Preferred subjects:

- Guest arrival.
- Room readiness.
- Stage/program environment.
- Professional coordination.
- Venue atmosphere.
- Hospitality details that support the story.

Avoid:

- Generic stock event photography.
- Confetti, champagne, balloons, or table-setting cliches as primary visuals.
- Private-event imagery without permission.
- Images that make the site feel like ProVia Events instead of Mahnaz's personal site.

## Graphic Language

Use subtle structure:

- Thin rules.
- Numbered proof blocks.
- Editorial pull quotes if real.
- Timeline markers.
- Calm card grids.

Avoid:

- Decorative swirls.
- Loud geometric patterns.
- SaaS blobs.
- Confetti effects.
- Overused event icons.

## Icon Direction

Use icons only when they improve scanning.

Style:

- Thin-line.
- Rounded but restrained.
- Single stroke weight.

Use for:

- Audience cards.
- Consultation topics.
- External trust links.

Avoid:

- Generic checklist overload.
- Calendar icon on every CTA.
- Decorative icons that repeat the headline.

## Visual Relationship To ProVia Events And EventAgent

ProVia Events:

- May be referenced through text links and small contextual brand mention.
- Do not import its full visual language directly.

EventAgent:

- May be referenced with structured/systems visual cues.
- Do not use software dashboard styling as the main look.
