# Stage 05 - Navigation Model

## Header Navigation

Recommended launch header:

- About
- Experience
- Event Strategy
- Book Consultation

`Book Consultation` should be styled as the primary CTA and link to Calendly.

## Header Rules

- Keep the header short.
- Do not include every audience segment in navigation.
- Do not include ProVia Events or EventAgent as primary nav items.
- Do not use a generic "Services" label at launch; it risks making the site feel like a service-company clone.
- Keep LinkedIn as an icon or secondary text link, not a primary CTA.

## Mobile Navigation

Mobile menu should include:

- About
- Experience
- Event Strategy
- Book Consultation
- LinkedIn

The Calendly CTA should remain easy to reach on mobile.

## Footer Navigation

Primary footer links:

- Home
- About
- Experience
- Event Strategy
- Book Consultation

Trust/context links:

- LinkedIn
- ProVia Events
- EventAgent

Utility links:

- Privacy, if required.

## Contextual Navigation

Use contextual links inside content:

- From About to Experience.
- From Experience to Event Strategy.
- From Event Strategy to Experience.
- From ProVia Events section to `https://proviaevents.com/`.
- From EventAgent section to `https://eventagent.io/`.
- From bio/credentials to LinkedIn.

## Navigation Anti-Patterns

Avoid:

- Services dropdowns.
- Separate navigation links for every audience.
- Leading with ProVia Events or EventAgent.
- Contact as the main CTA label.
- Blog/Insights in nav before content exists.
