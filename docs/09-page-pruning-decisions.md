# Stage 09 - Page Pruning Decisions

## Launch Pages Kept

Keep:

- Home.
- About.
- Experience.
- Event Strategy.

## Pages Deferred

Defer:

- Speaking & Emcee.
- Hospitality Consulting.
- Insights.
- Selected Work detail pages.

Reason:

- These are not required for the primary launch conversion path and need more content/assets before they earn space.

## Sections To Keep

Home:

- Hero.
- Proof strip.
- Who Mahnaz helps.
- Operating philosophy.
- Selected experience.
- Event Strategy Consultation.
- ProVia Events and EventAgent context.
- Final CTA.

About:

- Career arc.
- Approach.
- Credentials/proof.
- Context links.
- CTA.

Experience:

- Proof metrics.
- Event clusters.
- Selected examples.
- CTA.

Event Strategy:

- Problem framing.
- What the consultation covers.
- Best-fit scenarios.
- Preparation guidance.
- CTA.

## Sections To Remove Or Avoid

Avoid:

- Generic services grid.
- Blog teaser.
- Empty testimonials.
- Empty logo strip.
- Long generic process section.
- Multiple CTAs per audience.
- Stock event gallery.
- Placeholder Insights links.

## Implementation Pruning Rule

If a section does not improve trust, understanding, or action, remove it before visual QA.
